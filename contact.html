About AIXCC

AIXCC (AI eXploit Code Contest) is a groundbreaking competition that evaluates AI models' capabilities in identifying and exploiting security vulnerabilities in real-world code. The contest focuses on testing AI systems' understanding of security concepts, code analysis, and exploit development.
